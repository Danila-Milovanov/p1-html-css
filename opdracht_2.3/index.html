Onderzoek

Waarom noem je de eerste HTML-pagina index.html?

De eerste pagina van een website is meestaal heet als "index.html", omdat het de standaardpagina (waneer wordt geladen) wanneer je een website bezoekt zonder een specifiek pad op te geven.

Met welke HTML versie werken we nu?

We werken momenteel met HTML5

Met welke CSS versie werken we nu?

De huidige versie van CSS is CSS3.

Hoe kan het dat een website er net iets anders uitziet in een andere browser?

Elke browser heeft zijn eigen manier om HTML en CSS te openen, omdat ze hebben verschillende font's en instellingen.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Google Chrome is momenteel de populairste browser volgens Google zichzelf.
